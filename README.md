[c-honeypot](https://github.com/selmankon/c-honeypot) is the honeypot for this admin panel

## Usage
### Mac
```bash
source .venv/bin/activate
flask run --debug
```

### Windows
```bash
app\.venv\Scripts\activate
flask run --debug
```
